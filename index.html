Class representing a node in a binary tree

Attribute value

Attribute parent

the parent of the node, if any

Attribute left

the left child of the node, if any - assigning to this value will set the assigned Node’s parent

the right child of the node, if any - assigning to this value will set the assigned Node’s parent

__contains__(item)¶

Check if the directed edge or vertex is present in the graph.

A more Pythonic version of contains_edge() and contains_vertex().

To check a vertex, do vertex in dag. To check for an edge, do (a, b) in dag.

Returns

True if the directed edge or vertex is in the graph

__delitem__(item)¶

Remove an edge or vertex from the graph.

A more Pythonic version of remove_edge() and remove_vertex().

To delete a vertex, do del dag[vertex]. To delete an edge, do del dag[(a, b)].

__setitem__(v, adjacent)¶

Set the adjacency list for a vertex.

If the vertex is not present, it will be added to the graph

This allows you to create vertices and edges in a more Pythonic manner: - dag[1] = [] # inserts vertex 1 into the graph - dag[0] = [1, 2, 3] # creates edges (0, 1), (0, 2) and (0, 3)

Parameters

  • b – the ending vertex

Returns

the lowest common ancestor, if any
