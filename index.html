Software Engineering (CS3012)¶

This repo contains code and tests for the first assignment of the CS3012 module in Computer Science at Trinity College Dublin.

Implementation of the LCA (Lowest common ancestor) are provided for:

  • Binary trees

  • Directed acyclic graphs

Click the badges above for test results, documentation and coverage.

master contains the combined code of the binary tree and DAG implementations (along with their corresponding unit tests). tree contains only the binary implementation and dag is a branch where the DAG implementation was developed.
